Module 1 · OKRs
What is an OKR?
Before creating an OKR in KALÉ, let's understand what it is and what it's for.
💡
OKR stands for Objective and Key Results. It is a framework for defining goals and measuring whether they are being met.
  • Objective (O): A qualitative statement of what you want to achieve. E.g.: "Increase customer satisfaction in active projects"
  • Key Result (KR): A specific, measurable metric that indicates whether the objective is being achieved. E.g.: "NPS ≥ 70 in monthly survey"
  • Tasks: The concrete activities the consultant executes that contribute to moving a KR forward.

In KALÉ: each consultant has their own OKRs per period (Q1, Q2...). Tasks are linked to a specific OKR and KR, making it possible to know exactly whether the work creates value.

Create a new OKR
Click "+ New OKR" to open the creation form.
  • Click "+ New OKR" — a modal opens with the form
  • Write the Objective — it should be a clear phrase of what you want to achieve
  • Select the Responsible consultant from the list
  • Choose the Period (Q1, Q2, Q3 or Q4 · 2026)

Add Key Results
Key Results are the metrics that demonstrate whether you are achieving the objective.
  • Below the main form you will see the "Key Results" section with KR 1 already available
  • Write the KR description — it must be measurable and specific
  • Define the Target % — usually 100 (meaning you want to reach 100%)
  • If you need more KRs, click "+ Add another Key Result"
  • When done, click "Save OKR"

Module 1 · OKRs
Edit an OKR existente
You can modify any OKR at any time — KR progress is preserved.
  • In the OKR list, locate the OKR you want to modify
  • Click the "Edit" button — the same modal opens with pre-loaded data
  • Modify what you need: objective, consultant, period or Key Results
  • If you edit the text of an existing KR, the progress resets to 0% — be careful
  • Click "Save OKR" to confirm the changes

Módulo 2 · Tareas
Open "Link tasks"
Tasks are the concrete activities each consultant executes. Linking them to an OKR allows you to measure their impact.
  • In the navigation bar, click "Link tasks"
  • You will see the new task form at the top and the list of already created tasks below
  • Make sure you already have consultants created and at least one OKR before linking tasks

Módulo 2 · Tareas
Fill in the fields of la tarea
The form has two blocks: task identification and OKR linking.
  • Write the task name — be specific. E.g.: "Training workshop with logistics team"
  • Select the consultant responsible for executing it
  • Define the priority: High, Medium or Low
  • Select the linked OKR from the list — all created OKRs appear
  • Select the specific Key Result within that OKR
  • Define the start date (automatically filled with today) and the due date

Módulo 2 · Tareas
Status, progress and expected progress
This is the most important block — here the consultant reports their actual progress.
  • Select the Current status: Pending Start, In Progress, Blocked or Completed
  • Enter the Current progress % — it's a number between 0 and 100 that you define yourself
  • The Expected progress is calculated automatically — it is the percentage of days elapsed between the start date and due date
  • The system automatically shows whether you are on track (green), with a minor deviation (yellow) or delayed (red)
  • Click "Link and save" to save to Firebase

Módulo 2 · Tareas
Edit a task existente
You can update the status and progress of a task at any time.
  • In the "Linked tasks" section (below the form) locate the task you want to update
  • Click "Edit" — the form above pre-loads with all that task's data
  • Update the Status and Progress % according to your actual progress
  • Click "Save changes" — the button name changes automatically
  • Changes are reflected in the Dashboard and Value score in real time
